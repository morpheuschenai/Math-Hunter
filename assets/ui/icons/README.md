# Phase 2 UI Icon Pack

Purpose: shared chunky toy PNG icons for the Cute Chunky Math Quest UI direction.

Rules:
- Use these icons for UI surfaces before falling back to emoji.
- Keep icons 96x96 PNG with transparent background.
- Preserve thick black casing, bright toy fill, small white highlight, and bottom shadow.
- Add new files as `skill-<id>.png`, `reward-<id>.png`, or `nav-<id>.png`.
